Windows

Instalátor (EXE), 64-bit · 123 MB · Windows 10/11

Stáhnout MongeCAD-1.1.1-setup.exe
  • Instaluje se do profilu uživatele, nevyžaduje oprávnění správce.
Upozornění při instalaci: Instalátor momentálně není opatřen digitálním podpisem, takže Windows při spuštění pravděpodobně zobrazí varování „Systém Windows ochránil váš počítač“ (SmartScreen) nebo upozornění na neznámého vydavatele. Instalaci povolíte kliknutím na „Další informace“ a poté „Přesto spustit“. Podobné varování se může objevit i v prohlížeči při stahování — zvolte „Zachovat“ / „Přesto stáhnout“.

Windows — portable

Archiv ZIP, 64-bit · 149 MB · bez instalace

Stáhnout MongeCAD-1.1.1-portable_windows.zip
  • Rozbalte archiv a spusťte MongeCAD\bin\MongeCAD.exe.

Linux — Debian / Ubuntu

Balíček .deb, amd64 · 131 MB

Stáhnout mongecad_1.1.1_amd64.deb
  • Instalace: sudo apt install ./mongecad_1.1.1_amd64.deb

Linux — portable

Archiv ZIP, x86-64 · 155 MB · bez instalace

Stáhnout MongeCAD-1.1.1-portable_linux.zip
  • Rozbalte archiv a spusťte MongeCAD/bin/MongeCAD.
